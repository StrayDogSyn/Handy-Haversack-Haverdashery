# Pathfinder PDF Resources

This directory is for storing your Pathfinder PDF rulebooks and resources.

## Important Notes

⚠️ **Do NOT commit copyrighted PDF files to Git** - they are excluded in `.gitignore`

## Recommended Structure

```
pdfs/
├── core/
│   └── (Core Rulebook, Bestiary, etc.)
├── supplements/
│   └── (Advanced Player's Guide, etc.)
└── adventure-paths/
    └── (Campaign materials)
```

## Usage

Place your legally owned Pathfinder PDFs here. The application will eventually include features to:
- Parse monster/NPC stats from PDFs
- Extract spell information
- Import equipment and items
- Reference rules during gameplay

## Legal Notice

Only use PDFs that you have legally purchased or that are available through Paizo's Community Use Policy.
